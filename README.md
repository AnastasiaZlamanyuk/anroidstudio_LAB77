# Tip Time Theme — Калькулятор чаевых
## Цель работы
Познакомиться с понятием состояния **(state)** в Android-приложениях и научиться использовать его в интерфейсе с помощью **Jetpack Compose**. В результате лабораторной работы будет создано приложение **калькулятор чаевых (Tip Time)**, которое автоматически рассчитывает сумму чаевых. Изучить автоматизированные тесты в Android.
## Функциональность
Приложение реализует следующий набор возможностей:
- **Ввод суммы счёта** — числовое поле с меткой "Bill Amount" и иконкой валюты.
- **Ввод процента чаевых** — числовое поле с меткой "Tip Percentage" и иконкой процента.
- **Автоматический расчёт** — мгновенное вычисление суммы чаевых при изменении любого из полей.
- **Округление результата** — переключатель (Switch) для включения/отключения округления суммы чаевых вверх.
- **Форматирование валюты** — отображение результата в соответствии с локалью устройства (например, $, €, ₽).
- **Умная клавиатура** — настройка кнопок действия (Next → Done) для удобной навигации между полями.
- **Адаптивный интерфейс** — поддержка вертикальной прокрутки для корректного отображения в альбомной ориентации.
- **Локализация** — вынесение всех текстовых строк в ресурсы (strings.xml).
- **Тестирование** — реализованы локальные тесты для проверки логики расчёта и UI-тесты для проверки взаимодействия с интерфейсом.
## Технологии
Проект построен на следующих технологиях и библиотеках экосистемы Android:
- **Язык программирования**: Kotlin
- **UI-фреймворк**: Jetpack Compose
- **Material Design 3**: стилизация компонентов (TextField, Switch, Surface, темы)
- **Управление состоянием**: `mutableStateOf`, `remember`, механизм подъёма состояния (State Hoisting)
- **Ресурсы**: строковые ресурсы (`strings.xml`), аннотации `@StringRes` и `@DrawableRes`
- **Форматирование**: `NumberFormat` для локализованного отображения валюты
- **Тестирование**:
  - **JUnit 4** — локальные тесты бизнес-логики
  - **Compose UI Testing** — инструментальные тесты для проверки UI
- **Система сборки**: Gradle (Kotlin DSL)
- **Контроль версий**: Git, GitHub
